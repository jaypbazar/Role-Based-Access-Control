    Description

           Campus Parking Organizer is a web application that allows users to see available parking spaces inside the campus using real-time checking. It mainly focuses on effectively organizing vehicles parked inside the campus. This includes vehicles owned by students, faculties, and other employees. The main features of this app include real-time checking, map navigation, and space reservation.

    Features

    1. Real-time Checking - users can always check the number of available parking spaces inside the campus using the indicator above the interactive map.
    2. Map Navigation - users can view the locations of different parking spaces inside the campus using the interactive map. Green spaces indicates available parking space, Red indicates occupied, and Blue indicates reservation.
    3. Parking Space Reservation - only users that are currently logged in can reserve a particular parking space.
